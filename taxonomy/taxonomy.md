# Guttenberg — Taxonomy

Reserved terms for the game. Each term has a canonical definition. If a term appears in game content, it means exactly what it says here — nothing more, nothing less.

---

## Session Types

### Headless Session
A game session that runs automated — like a batch job, overnight process, or background task — without human intervention. The engine drives all decisions, rolls, and narrative progression autonomously. Contrast with a live session where a human player or GM is actively participating.
